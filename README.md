# CrossmeSolution
- Crossme是一个Nonogram类解谜益智游戏，本仓库将以Crossme的App Store版本为准，按照其中章节和编号列出每个迷题的答案
- 欢迎各位喜欢Nonogram类游戏的朋友积极提供答案
- 迷题答案使用截图工具进行截取后收录到对应迷题目录中
- Origin为普通黑白色迷题的所有答案
- Color为彩色迷题的答案
